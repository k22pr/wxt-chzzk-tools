# Chzzk Tools

치지직(CHZZK) 방송 시청과 관련된 유틸리티를 제공하는 브라우저 확장 프로그램입니다.

## 주요 기능

### 🎬 기능 탭

#### 자동 화질 변경

- 그리드를 설치하지 않아도 **1080p**로 자동 재생합니다.
- 광고 차단 프로그램 사용 시 화질이 낮아지는 문제를 해결합니다.

#### LIVE 재생바

- 라이브 방송 접속 시점부터 **최대 1분 30초**간 되감기가 가능한 재생바를 제공합니다.
- 놓친 장면을 간편하게 다시 볼 수 있습니다.

#### 음향 보정 (Compressor)

- 오디오 컴프레서를 통해 소리를 **증폭**하거나 **조절**할 수 있습니다.
- 볼륨/음질을 조절할 수 있는 오버레이를 표시합니다.
- 채널별로 설정이 자동 저장됩니다.

#### 팔로잉 자동 새로고침

- 사이드바의 팔로잉 목록을 **30초마다** 자동으로 새로고침합니다.
- 팔로우한 스트리머의 방송 시작을 빠르게 확인할 수 있습니다.

#### VOD 실제 시간

- VOD 탐색 시 마우스를 올리면 **실제 방송 시간**을 툴팁으로 표시합니다.
- 특정 시점의 실제 시간을 손쉽게 확인할 수 있습니다.

### 🎨 모양 탭

#### 동영상 재생바 스타일 변경

- 동영상 재생바에 **블러(Blur) 디자인**을 적용합니다.
- 프리미엄한 느낌의 플레이어 컨트롤바를 제공합니다.
- PIP(미니 플레이어) 모드에서는 기본 디자인이 유지됩니다.

## 설치 방법

1. 릴리즈 페이지에서 최신 버전을 다운로드합니다.
2. Chrome에서 `chrome://extensions`로 이동합니다.
3. 개발자 모드를 활성화합니다.
4. "압축해제된 확장 프로그램을 로드합니다"를 클릭하고 압축 해제한 폴더를 선택합니다.

## 기술 스택

- [Vue 3](https://vuejs.org/) - UI 프레임워크

- [UnoCSS](https://unocss.dev/) - CSS 유틸리티
