Pzzk Tools

라이브 스트리밍 방송 시청을 더욱 편리하게 만들어주는 브라우저 확장 프로그램입니다.

Chrome 웹 스토어

주요 기능

자동 화질 변경

광고 차단 프로그램 사용 시 화질이 낮아지는 문제를 해결하고, 자동으로 1080p로 재생합니다.

LIVE 재생바

라이브 방송 접속 시점부터 최대 1분 30초간 되감기가 가능한 재생바를 제공합니다.

음향 보정

오디오 컴프레서를 통해 소리를 증폭하거나 조절할 수 있습니다. 채널별로 설정이 자동 저장됩니다.

팔로잉 자동 새로고침

사이드바의 팔로잉 목록을 30초마다 자동으로 새로고침합니다.

VOD 실제 시간

VOD 탐색 시 마우스를 올리면 실제 방송 시간을 툴팁으로 표시합니다.

프리미엄 UI

동영상 재생바에 세련된 디자인을 적용하여 더 나은 시청 경험을 제공합니다.
