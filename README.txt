Chzzk Tools

주요 기능
=========

[기능 탭]

자동 화질 변경
- 그리드를 설치하지 않아도 1080p로 자동 재생합니다.
- 광고 차단 프로그램 사용 시 화질이 낮아지는 문제를 해결합니다.

LIVE 재생바
- 라이브 방송 접속 시점부터 최대 1분 30초간 되감기가 가능한 재생바를 제공합니다.
- 놓친 장면을 간편하게 다시 볼 수 있습니다.

음향 보정 (Compressor)
- 오디오 컴프레서를 통해 소리를 증폭하거나 조절할 수 있습니다.
- 볼륨/음질을 조절할 수 있는 오버레이를 표시합니다.
- 채널별로 설정이 자동 저장됩니다.

팔로잉 자동 새로고침
- 사이드바의 팔로잉 목록을 30초마다 자동으로 새로고침합니다.
- 팔로우한 스트리머의 방송 시작을 빠르게 확인할 수 있습니다.

VOD 실제 시간
- VOD 탐색 시 마우스를 올리면 실제 방송 시간을 툴팁으로 표시합니다.
- 특정 시점의 실제 시간을 손쉽게 확인할 수 있습니다.


[모양 탭]

동영상 재생바 스타일 변경
- 동영상 재생바에 디자인을 적용합니다.
- PIP(미니 플레이어) 모드에서는 기본 디자인이 유지됩니다.
